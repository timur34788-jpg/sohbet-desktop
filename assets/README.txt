İKON DOSYALARI

Build için şu dosyaları buraya ekle:

  icon.ico   → Windows için (256x256 px)
  icon.icns  → macOS için
  icon.png   → Linux için (512x512 px)

Şu an bu dosyalar olmadan da build alınabilir,
Electron varsayılan ikonu kullanır.

İkon oluşturmak için: https://www.electronjs.org/docs/latest/tutorial/application-distribution
